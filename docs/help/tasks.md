# Tasks

A task is the smallest unit tinstar tracks. Sessions can attach to tasks; commits tagged `#<task-name>` show up in the task's activity panel.

## Commands

```
tinstar tasks list
tinstar tasks create "Add onboarding wizard" <epicId>
```
