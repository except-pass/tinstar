# Onboarding

Tinstar's empty state walks you through four steps:

1. **Connect** to a backend (Tauri only — browser users skip this).
2. **Create a workspace** — top-level container.
3. **Register a project** — the git repo your sessions will run inside.
4. **Start your first session** — a CLI agent attached to the project.

Run `tinstar status` any time to see what's missing.
