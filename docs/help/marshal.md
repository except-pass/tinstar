# Marshal

Marshal is a tinstar session that runs by default in the canvas sidebar. Ask it open-ended questions about tinstar; it uses `tinstar help` to look things up.

## When to use marshal

- "What's an epic?" — concept questions.
- "What should I do next?" — discovery.
- "Spawn three reviewers on this task" — control-plane ops.

## When to use the CLI directly

- Scripted operations.
- Repeatable admin tasks.
